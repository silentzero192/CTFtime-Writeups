# CTFtime Writeups

This repository contains my personal writeups for Capture The Flag (CTF) challenges solved during competitions listed on CTFtime.

The purpose of this repository is to:

- Document complete challenge analysis and solution methodologies
- Preserve technical notes for future reference
- Share knowledge with the security community
- Strengthen understanding of cryptography, binary exploitation, forensics, web security, reversing, and related domains

---

## Repository Structure
CTFtime-Writeups/
│
├── 2025/
├── 2026/
└── README.md


Each yearly directory contains writeups organized by competition name and category.

## What You’ll Find Here

- Detailed breakdown of challenge logic
- Vulnerability analysis
- Mathematical reasoning (where applicable)
- Exploitation methodology
- Full working scripts or proof-of-concept solutions
- Lessons learned from each challenge
